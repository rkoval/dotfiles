# rkoval's dotfiles

- Restart your Mac while pressing Cmd+R to boot into recovery OS and run:

  ```sh
  csrutil disable
  reboot
  ```

- Install XCode from App Store
- Explicitly open XCode after install to install extras
- Initialize dotfiles:

  ```sh
  ./init.sh
  ```

- Restart your Mac while pressing Cmd+R to boot into recovery OS and run:

  ```sh
  csrutil enable
  reboot
  ```
